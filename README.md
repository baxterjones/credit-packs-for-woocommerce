# Lesson Credit Wallet

A lightweight WordPress plugin that adds prepaid lesson credits to WooCommerce appointment bookings.

Designed for tutoring businesses that sell lesson packs instead of requiring payment for every booking.

---

## Features

- Credit Pack products
- Redeemable lesson products
- Tutor LMS dashboard integration
- Credit ledger
- Manual credit adjustments
- WooCommerce HPOS compatible
- Offline / EFT workflow
- Full refund support
- Product validation

---

## Requirements

- WordPress 6.7+
- WooCommerce
- Tutor LMS
- WooCommerce Appointments

---

## Installation

1. Upload the plugin.
2. Activate it.
3. Create Credit Pack products.
4. Create Redeemable Lesson products.
5. Start selling lesson packs.
